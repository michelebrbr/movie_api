Allow new users to register /users POST A JSON user object. Example:
{
Username: 'leorio11',
Password: 'medicine',
Email: 'leoriohxh@mockgmail.com',
Birthday: 11-08-1996
}
 A text message indicating wheter the user was successfully registered
Allow users to update their user info /users/:username PUT A JSON user object. Example:
{
Username: 'leorio11',
Password: 'medicine',
Email: 'leoriohxh@mockgmail.com',
Birthday: 11-08-1996
}
 A text message indicating wheter the user info was successfully updated
Allow users to add a movie to their favorites /users/:username/movies/:_id(MovieID) PUT The username of the user and the MovieID (_id) A text message indicating wheter the movie was successfully added to the user favorite list
Allow users to remove a movie from their list of favorites /users/:username/movies/:_id(MovieID) DELETE The username of the user and the MovieID A text message indicating whether the movie was successfully removed
Allow existing users to deregister /users/:username DELETE The user's username A text message indicating whether the user was successfully deregistered
